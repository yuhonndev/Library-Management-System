# Library-Management-System

## INTRODUCTION 
Lumina Library System makes it easier to keep track of books, handle members, while streamlining checkouts. Created for speed and simplicity, this tool lets libraries sort their materials without hassle  helping staff work better, so visitors get quicker service.

The tools we used to build the system

* Java
* NetBeans IDE
* XAMPP/MYSQL WORKBENCH/LARAGON

## SYSTEM FEATURES

* LOADING PAGE
* LOGIN PAGE (REGISTRATION, ONLY LIBRARIAN CAN ACCESS THIS SYSTEM)
* HOME PAGE
* NEW STUDENT PAGE
* OUTSIDER PAGE
* BOOK ( IF AVAILABLE OR NOT)
* BORROWING PAGE 
* LIBRARIAN PAGE
* RETURN PAGE
